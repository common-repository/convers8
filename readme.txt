Enable social network login with Facebook, Hyves, Twitter and LinkedIn on your Wordpress blog.
== Description ==

This plugin enables users to login and connect one or more social network accounts to your Wordpress blog.
We support the following networks: Facebook, Hyves, Twitter and LinkedIn.
After installation, you have to register for a Convers8 API key also at [convers8.eu](http://convers8.eu/ "Convers8.eu").

== Installation ==

1. Install and enable the Convers8 plugin on your Wordpress website
2. Request a Convers8 API key at [convers8.eu](http://admin.convers8.eu/wizard/ "Convers8.eu")
3. Fill in the Convers8 Website ID, Convers8 Website Key and Convers8 URL at the plugin settings. 
You can find the plugin settings in your Wordpress Blog admin site -> Settings -> Convers8 
4. If you prefer, activate the Convers8 connect widget through Wordpress Blog admin site -> Display -> Widgets
